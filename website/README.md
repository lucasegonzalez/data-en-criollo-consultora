# Landing Page — Cloudflare Pages

This static site is deployed via **Cloudflare Pages** for the consulting landing page.

- **Domain**: `dataencriollo.com` (or your custom domain)
- **Framework**: None (plain HTML/CSS)
- **Deploy**: Connect this repo to Cloudflare Pages → point to `website/` directory

## How to deploy

1. Go to [Cloudflare Dashboard → Pages](https://dash.cloudflare.com/)
2. Create a new project → Connect your Git repository
3. Build settings:
   - **Build command**: (leave empty — static HTML)
   - **Build output directory**: `website`
4. Deploy!
